# Static UI Handoff

فایل‌های این فولدر نمونه‌ی ایستای صفحات هستند تا تیم اندروید بتواند بدون اجرا کردن Next.js، طراحی را مشاهده کند.

ساختار:
- `dashboard.html` — نمونه صفحه داشبورد/ویزیت‌ها با کارت و جدول
- `customers.html` — لیست مشتریان نمونه
- `invoices.html` — پیش‌فاکتورها
- `assets/styles.css` — استایل پایه (در صورت نیاز با build نهایی جایگزین کنید)
- `assets/scripts.js` — تعاملات ساده + بارگذاری mock data
- `mock-data/` — داده‌ی نمونه برای جداول و کارت‌ها (JSON)

دستور build (در ریشه پروژه):
```bash
npm run handoff:build
```
که یک نسخه جدید از CSS در این فولدر تولید می‌کند.

> **نکته:** برای اینکه فایل‌های JSON به‌درستی بارگذاری شوند، این فولدر را با یک وب‌سرور ساده (مثل `npx serve handoff/static-ui`) اجرا کنید. مرورگر در حالت `file://` اجازه‌ی `fetch` به فایل‌های محلی را نمی‌دهد.
