# مستندات پروژه تعمیربان

این پوشه برای نگهداری مستندات تکمیلی استفاده می‌شود. فایل‌های پیشنهادی در فازهای بعدی عبارت‌اند از:

- `deployment-cpanel.md` — راهنمای استقرار روی سی‌پنل و مدیریت `server.js`
- `data-model.md` — توضیح مدل‌های داده، اسکیمای MongoDB و ارتباطات
- `architecture.md` — تصمیمات معماری و ساختار ماژول‌ها
- `flutter-handoff.md` — تحویل به تیم Flutter/Android پس از فازهای وب

لطفاً پس از ایجاد هر فایل، `todo.md` را متناسب با تغییرات به‌روزرسانی کنید.
